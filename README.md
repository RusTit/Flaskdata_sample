# Flaskdata sample

![Python package](https://github.com/RusTit/Flaskdata_sample/workflows/Python%20package/badge.svg)

## Installation

```shell
pip install -r requirements.txt
```

## Configuration

Please set up environment variables

```dotenv
FLASK_API_USER
FLASK_API_PASSWORD
FLASK_API_IDP_USER
FLASK_API_IDP_PASSWORD
```

## Usage

```shell
python main.py
```
